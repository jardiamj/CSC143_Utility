# Utility tools for Computer Science class CSC143 - Java II

- xlint: bash script for running the java compiler (javac) with required -Xlint
options (unchecked,rawtypes) and -Xdoclint:all on all .java files that are not
JUnit test classes.

- bluej_gitignore: gitignore file for excluding tracking of common generated
files when coding on java using the BlueJ IDE.
